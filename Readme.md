# DevPilot System Inspector

A professional Python-based system inspection toolkit that analyzes hardware, operating system, installed software, package managers, network interfaces, and overall system health through a rich terminal dashboard.

![Python](https://img.shields.io/badge/Python-3.10+-blue)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)
![License](https://img.shields.io/badge/License-MIT-green)

---

## Features

- System Information
- CPU Monitoring
- Memory Analysis
- Disk Usage & I/O Statistics
- Network Interface Inspection
- Installed Software Detection
- Package Manager Detection
- Health Analysis
- Rich Terminal Dashboard
- Modular SDK Architecture

## Installation

```bash

cd Windows-System-Inspector

python -m venv .venv

.venv\Scripts\activate

pip install -r requirements.txt
```
